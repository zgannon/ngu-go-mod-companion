# Security Policy

## Supported Versions

This project is currently in early development. Security fixes are applied to the latest version on the default branch.

## Reporting a Vulnerability

Please report vulnerabilities privately by opening a private security advisory on GitHub for this repository.

If private advisories are not available, open an issue with minimal details and request a secure contact channel before sharing exploit details.

## Response Goals

- Initial acknowledgement: within 7 days
- Status update after triage: within 14 days
- Fix timeline: depends on severity and reproduction details
